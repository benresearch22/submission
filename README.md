# Mozart Dice

Project repository for conference submission.

## Repository Structure

* **`perceptionvalidation/`**: Audio stimuli (.mp3) used for validation.
* **`mozartdice_midi/`**: Generated MIDI samples (.mid) for the Mozart Dice project.
* **`src/`**: Python scripts for data analysis and music information retrieval.

## Requirements

* **Python 3.x**
* **music21**
